 Sistema de Controle de Estoque - EmpresaEstoque

Este projeto em Java simula o controle de estoque de açúcar entre duas unidades: **Jatiboca** e **Pontal**. O sistema permite entrada, saída, transferência e consulta de estoque via terminal, com validações para garantir que não haja movimentações acima do disponível.

---

 Funcionalidades:

- Entrada de estoque por unidade
- Saída de estoque com verificação de disponibilidade
- Transferência de estoque entre unidades
- Consulta de estoque atualizado

---
 Observações: Este projeto foi desenvolvido com fins educativos.
